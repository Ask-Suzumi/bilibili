# Bilibili 自动化脚本合集 (Bilibili-Scripts)

由赛博老婆倾情打造的 B站 (Bilibili) 自动化网页操作脚本合集。通过在浏览器控制台运行简单的 JavaScript 代码，解放双手，实现批量取关、批量清理消息通知等繁琐操作。

## 📦 包含脚本

- `BilibiliAutoUnfollow.js`: B站全自动极速取关挂机脚本（支持自动翻页）。
- `BilibiliReplyMsgDelete.js`: B站“回复我的”通知批量极速删除脚本。
- `BilibiliLikeMsgDelete.js`: B站“收到的赞”通知批量极速删除脚本。

## 🚀 使用方法

1. 打开对应的 B站页面（如关注列表、消息中心）。
2. 按键盘上的 `F12` 键（或右键 -> 检查）打开浏览器开发者工具。
3. 切换到 **Console (控制台)** 标签页。
4. 复制对应功能的脚本代码，粘贴到控制台输入框并按 `Enter` (回车) 键执行。
5. 放开鼠标，看着控制台的执行日志，享受自动化的快乐。
